#**Trippin'** (work in progress)

###About the App
Allows you to create your own blog documenting moments on a specific trip. After you create an account, you will be able to make a new trip and add posts to that particular trip.

###Technologies:
* Ruby on rails
* HTML
* CSS

###Tools to help plan:

* Simple wireframe
